Privacy Policy

Last updated: 2025-09-30

Y. Music Exporter is a browser extension that exports visible track lists from music.yandex.ru to local files. The extension works entirely on the user’s device. We do not collect, store, sell, or transmit any personal data.

Data collection & transfer

  • No data is sent to external servers or third parties.
  • No analytics, telemetry, ads, or profiling.
  • Track information is read from the currently open page solely to generate your export file.

Permissions explained

  • Host permissions (https://music.yandex.ru/*): required to read visible track items on that site only.
  • activeTab/tabs: used to address the current tab and communicate with the content script. No browsing history is accessed.
  • scripting: injects a bundled content script to parse the page. No eval, no remote scripts.
  • storage: keeps temporary in-extension state (e.g., list before export). Nothing is persisted after the session.
  • downloads: saves TXT/TSV files locally when you click “Export”.

Remote code

The extension does not use remote code or load external JavaScript/Wasm. All code ships within the package.

Children’s privacy

This extension does not target children and does not process personal data.

Third parties

No third-party services are integrated. Exported files stay on your device.

Changes to this policy

If the policy changes, we will update the “Last updated” date above. Substantive changes will be reflected in the published version of the extension and this page.
